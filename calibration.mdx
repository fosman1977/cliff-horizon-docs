Calibration is necessary but not sufficient for the moat. The moat is forward-climate pricing × breadth × network. Calibration earns the right to price — it does not, by itself, create the competitive advantage.

---

## Governance rule

> **A calibration change ships only if it lowers out-of-sample CRPS without degrading ECE or PIT.**

This is the single gate. No exceptions. The out-of-sample validation period is **2015–2024** (held out from the 1991–2014 calibration period). A change that improves in-sample fit but degrades OOS performance is not shipped.

---

## Metrics

| Metric | What it measures | Target |
|---|---|---|
| **CRPS** | Continuous Ranked Probability Score — proper scoring rule for full distributions | Minimise |
| **ECE** | Expected Calibration Error — weighted average gap between forecast probability and observed frequency | < 0.03 |
| **PIT histogram** | Probability Integral Transform — should be uniform if model is calibrated | Flat |
| **ERA5 ↔ NASA basis** | Independent-truth basis at each site — quantifies settlement uncertainty | Disclosed per site |

CRPS is the primary gate. ECE and PIT are secondary — a change that passes CRPS but introduces systematic over/under-confidence in the reliability diagram is not shipped.

## The 6 calibration levers

| Lever | Description |
|---|---|
| **M0** | Baseline — quantile mapping (QM), monthly scale factor |
| **M1** | Truth-layer fusion — pin settlement gauge; ERA5→IMD re-targeting |
| **M2** | QDM (Quantile Delta Mapping) — trend-preserving bias correction |
| **M3** | CRPS-tuned recalibration (EMOS) — ensemble model output statistics |
| **M4** | Seasonal regime separation — monsoon vs dry season models |
| **M5** | CMIP6 ensemble loading — forward-climate distribution update |
| **M6** | Multi-site pooling — borrow strength across correlated sites |

## Milestone table

| Milestone | Status | CRPS | ECE | Notes |

| M0 Baseline (QM monthly) | Done | 1.655 | 0.030 | PIT non-uniform: wet-tail under-dispersion |
| M1 Truth-layer fusion | In progress | — | — | ERA5 wet outlier vs IMD; gated ERA5→IMD re-target next |
| M2 QDM (trend-preserving) | Shipped | 1.649 | 0.025 | PASS — both metrics improved |
| M3 CRPS-tuned (EMOS) | Not shipped | No improvement | — | CRPS did not improve OOS; rejected |
| M4–M6 | Queued | — | — | Behind M1 |

**M2 is the current production method.** CRPS improved from 1.655 → 1.649 and ECE from 0.030 → 0.025. Both are statistically significant at the 90% level based on bootstrap CI.

**M3 was a negative result.** EMOS (Ensemble Model Output Statistics) is a standard recalibration method. It improved in-sample but did not improve OOS CRPS. It was not shipped. The governance rule held.

## How to read the proof harness

The `/api/proof/{site_id}` endpoint is the live read-out of the calibration state for a given site. Key things to check:

**Is CRPS improving?** Compare to the M0 baseline (1.655). Any production model should be below this.

**Is the reliability diagram on the diagonal?** Points systematically above or below the diagonal indicate mis-calibration. The current M2 QDM model shows good reliability except at the upper tail (>80th percentile).

**Is the PIT histogram flat?** Right-skew in the PIT histogram means the model under-disperses in the wet tail — it assigns too-low probability to extreme events. This was the M0 diagnosis and is partially addressed by M2 QDM.

**Is ECE below 0.03?** At 0.025, M2 passes this threshold with margin.

## The calibration / moat relationship

Calibration earns trust. It is the condition for a buyer or capital counterparty to accept CH's numbers as a settlement basis. But it is not the moat because:

1. Any quant team can run QDM on ERA5
2. The improvement in CRPS from M0 to M2 (1.655 → 1.649) is small — it matters at the margin, not as a decisive differentiator
3. The data sources (ERA5, NASA POWER, GPM IMERG) are free and public

The moat comes from combining calibration with breadth (many uncorrelated perils/geographies) and network (buyer + capital relationships that compound). Calibration alone is table stakes.
